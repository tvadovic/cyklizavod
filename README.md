# Cyklistický Závod / náhradná aplikácia ku hre Tour de France Board Game (https://www.cycling-game.com/en/tour-de-france-board-game)

Digitálna časomiera a výsledkový systém pre cyklistické závody.

Ideálne na hranie s figúrkami – každý jazdec má pevné číslo a farbu tímu.

## Použitie

1. Otvor https://tvadovic.github.io/cyklizavod/
2. Klikni **START GAME**
3. Vyber režim (etapový závod / jednorázový závod)
4. Nastav tímy a jazdcov
5. Losuj štartové poradie a začni preteky

## Ovládanie počas etapy

- **TURN** – posunie čas o +1 minútu
- **SPRINT** / **MOUNTAIN** – prideľ body za prémie (5-3-2-1)
- **FINISH** – zadaj jazdcov v cieli (čas + bonus prvé 3 miesta)
- **RESULTS** – priebežné výsledky

## Výsledky

Po každej etape:
- Poradie jednotlivcov (žltý dres)
- Sprintérska súťaž (zelený dres)
- Vrchárska súťaž (bodkovaný dres)
- Tímová klasifikácia (3 najlepší z každého tímu)

## PWA / Android

Aplikáciu je možné nainštalovať ako samostatnú appku na Android v Chrome → Pridať na plochu alebo inštalovať.
